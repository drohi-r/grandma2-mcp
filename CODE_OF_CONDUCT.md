# Code of Conduct

This project adopts the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/) as its code of conduct.

All contributors, maintainers, and participants are expected to uphold these standards in all project spaces — GitHub issues, pull requests, discussions, and any other community venue.

## Our Pledge

We are committed to making participation in this project a welcoming experience for everyone, regardless of background, experience level, or identity.

## Reporting

Instances of unacceptable behavior may be reported to the project maintainer. All reports will be reviewed promptly and kept confidential.

For the full text of the Contributor Covenant v2.1, see:
https://www.contributor-covenant.org/version/2/1/code_of_conduct/
